# Task 3

Downloaded a CSV file and added it to the GitHub_task1 folder. Also created a new branch called Temp and performed all operations in that branch.

## Linux Commands Used:
- cp ~/Downloads/<file>.csv ~/GitHub_task1/  
  Copies CSV file from Downloads to project folder.

## Git Commands Used:
- git checkout -b Temp  
  Creates and switches to a new branch named Temp.

- git add .  
  Stages all files including the CSV file.

- git commit -m "Added CSV file"  
  Saves changes in the Temp branch.

- git push origin Temp  
  Pushes the Temp branch to GitHub.
