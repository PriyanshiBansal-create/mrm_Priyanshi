# Task 4 & Task 5

## Task 4:
Created a new branch named Temp and performed operations in that branch.

## Task 5:
Merged the Temp branch into the main branch.

## Git Commands Used:

- git checkout -b Temp  
  Creates and switches to a new branch.

- git add .  
  Stages files in Temp branch.

- git commit -m "message"  
  Saves changes in Temp branch.

- git push origin Temp  
  Pushes Temp branch to GitHub.

- git checkout main  
  Switches back to main branch.

- git merge Temp  
  Merges Temp branch into main branch.
